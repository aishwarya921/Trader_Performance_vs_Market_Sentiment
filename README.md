# Trader Performance vs Market Sentiment

## Overview
This project analyzes how trader performance and behavior change under different market sentiment regimes (Fear vs Greed).

## Methodology
- Cleaned and aligned trader and sentiment data by date
- Created daily metrics such as PnL, win rate, and trade count
- Compared trader behavior across Fear and Greed periods
- Segmented traders based on activity and risk
- Built a simple predictive model

## How to Run
1. Open the Jupyter notebook
2. Run all cells from top to bottom

## Tools Used
Python, Pandas, Matplotlib, Scikit-learn
